# E-Commerce Operations & Returns Dashboard


## Project Overview
This project analyzes e-commerce operational performance with a focus on shipping efficiency, service levels, and return risk.

## Business Problem
How can delivery performance be optimized while identifying product categories that contribute most to return risk?

## Key Insights
- Average shipping time improved after 2017 and stabilized near 4 days.
- Shipping mode selection is the primary driver of delivery speed.
- Return risk is highest in Technology and Furniture categories.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Deliverables
- Interactive Power BI dashboard
- KPI tracking for shipping and returns
- Category-level return risk analysis
