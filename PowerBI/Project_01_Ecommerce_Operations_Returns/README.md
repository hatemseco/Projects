# E-Commerce Operations & Returns Dashboard

## Overview
This Power BI dashboard analyzes e-commerce operational performance with a focus on shipping efficiency, service levels, and return risk. The goal is to identify operational bottlenecks and product categories that drive higher return rates.

## Key KPIs
- Returned Orders
- Return Rate (%)
- Average Shipping Days
- Orders Shipped Within 2 Days (%)

## Key Insights
- Shipping efficiency improved over time, with average delivery days declining below 4 days.
- Ship mode is a major driver of delivery speed, with Same Day and First Class significantly outperforming Standard Class.
- Return risk is concentrated in Technology and Furniture categories, indicating potential issues with product expectations or handling.
- Overall service reliability remains strong, with a meaningful portion of orders delivered within two days.
