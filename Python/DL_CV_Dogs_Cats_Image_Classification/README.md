# Dogs & Cats Image Classification 

## Overview
This project focuses on building and evaluating machine learning and deep learning models to classify images of 


**dogs and cats**. The work explores both traditional machine learning baselines and convolutional neural networks to understand how different modeling approaches perform on image data.



## Dataset
- Dogs & Cats image dataset
- Images processed as RGB images
- Image size standardized during preprocessing
- Binary classification task:
  - Dog
  - Cat

---

## Objectives
- Load and preprocess image data for modeling
- Establish baseline machine learning models for comparison
- Build and train deep learning models for image classification
- Evaluate model performance using appropriate metrics
- Analyze results and identify areas for improvement

---

## Work Completed

### Data Preparation
- Loaded image data using Python
- Converted images into numerical arrays
- Normalized pixel values
- Resized images for consistent input shape
- Split data into training and validation sets

---

### Machine Learning Models
- Implemented baseline models to establish reference performance
- Trained classifiers on processed image data
- Evaluated models using:
  - Accuracy
  - Confusion matrix

---

### Deep Learning
- Built a Convolutional Neural Network (CNN)
- Implemented:
  - Convolution layers
  - Pooling layers
  - Fully connected layers
- Trained models using gradient-based optimization
- Evaluated model performance on validation data

---

### Evaluation
- Generated confusion matrices to analyze classification results
- Compared performance across different modeling approaches
- Identified strengths and weaknesses of each model type
