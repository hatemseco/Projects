# Fashion MNIST Classification 

---

## Overview
This project focuses on classifying fashion product images using the **Fashion MNIST** dataset. The objective is to build and evaluate machine learning and deep learning models capable of distinguishing between multiple clothing categories based on grayscale image data.

The project demonstrates a complete image classification workflow, from data preprocessing to model training and evaluation.

---

## Dataset
- Fashion MNIST dataset
- Grayscale images of fashion items
- Image size: 28 Ã— 28 pixels
- Multi-class classification task with 10 categories:
  - T-shirt/top, Trouser, Pullover, Dress, Coat
  - Sandal, Shirt, Sneaker, Bag, Ankle boot

---

## Objectives
- Load and preprocess image data for modeling
- Build classification models for multi-class image recognition
- Train deep learning models on image data
- Evaluate model performance across multiple classes
- Analyze classification results and error patterns

---

## Data Preparation
- Loaded image data using Python
- Normalized pixel values
- Reshaped image arrays for model compatibility
- Split data into training and validation sets

---

## Machine Learning & Deep Learning
- Implemented classification models for multi-class prediction
- Trained neural network models on image data
- Optimized model parameters to improve accuracy
- Evaluated model performance using validation data

---

## Evaluation & Analysis
- Measured overall classification accuracy
- Generated confusion matrices to analyze class-level performance
- Identified commonly confused clothing categories
- Evaluated strengths and limitations of the trained models
